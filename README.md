# React SPA Application

A SPA application for working with data from the DummyJSON API using a modern technology stack.

## Features

- **React 18** with hooks
- **React Router v6** for routing
- **Redux Toolkit** for state management
- **RTK Query** for caching and working with APIs
- **Mantine UI** for UI components
- **Webpack 5** for bundling (without Create React App)
- **Responsive design** for mobile and desktop

## Pages
1. **Login** (`/login`) 
2. **Register** (`/register`) 
3. **Product list** (`/products`)
4. **Product Details** (`/products/:id`) 

## Technical Features
- User Authorization
- Pagination and Search
- Data Filtering
- Request Caching
- Responsive Layout
- Error Handling
- State Loading
